1. 수집하는 개인정보

「후다닥 미세먼지 피하기」(이하 "서비스")는 대기질 정보를 제공하기 위해 아래 정보를 수집합니다.

  • 위치정보(위도·경도) — 사용자 주변의 대기질 측정소를 찾기 위한 목적

2. 위치정보의 이용 목적

  • 사용자와 가장 가까운 대기질 측정소를 탐색하여 미세먼지(PM10), 초미세먼지(PM2.5) 및 기타 공기지표(SO₂, CO, O₃, NO₂)를 제공

3. 위치정보의 보유 및 파기

수집된 위치정보는 서버에 저장하지 않으며, 요청 처리 즉시 폐기됩니다. 브라우저 세션이 종료되면 클라이언트 측 위치 데이터도 소멸합니다.

4. 제3자 제공

위치정보는 아래 서비스에 대기질 조회 목적으로만 전달됩니다.

  • hudadak-air API — 자체 DB 조회 및 역지오코딩
  • Kakao 로컬 API — 주소 검색 및 역지오코딩 보조
  • Open-Meteo API — 미세먼지 모델 예측

상기 목적 이외에 위치정보를 제3자에게 제공하거나 판매하지 않습니다.
