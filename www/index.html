후다닥 미세먼지 v5.1 변경사항

변경일: 2026.07

후다닥 미세먼지의 대대적인 내부 업데이트가 있었습니다. 사용자에게 보이는 화면보다 데이터 품질과 안정성 개선에 집중한 버전입니다.

데이터 & 백엔드

  • 다중 데이터 소스 통합 엔진: WAQI 실측 + Open-Meteo 모델 + OpenAQ 집계를 하나로 여어 관리. 실측 없는 지역은 모델 예측값으로 자동 전환하여 데이터 공백을 최소화했습니다.
  • 고정된 지역명 매핑 방식에서 보어, 현재 위치의 위경도를 기준으로 DB에서 가장 가까운 측정소를 실시간 탐색하는 방식으로 전환했습니다.

기능 추가

  • 홈 화면 위젯 (Android): 앱을 열지 않아도 홈 화면에서 현재 위치의 미세먼지 수치를 바로 확인할 수 있습니다.
  • 설정 메뉴: 다크 모드 / 라이트 모드 전환, 위젯 사용 여부를 앱 내 설정에서 관리할 수 있습니다.

개인정보 & 운영

  • 카카오톡 채널 @soulspectrum을 개설했습니다. 앱 관련 문의는 카카오톡 채널로 통합 운영합니다.
  • 운영사 (주)소울스펙트럼의 정식 사업자 정보를 앱 내에 공개합니다.
