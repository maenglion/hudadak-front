후다닥 미세먼지 v5.1 변경사항

변경일: 2026.07

이번 v5.1은 사용자에게 보이는 화면보다 데이터 품질과 서비스 안정성 개선에 집중한 버전입니다.

데이터 & 백엔드

  • 측정소 DB 전면 리팩토링
    기존에는 고정된 측정소 목록에서 데이터를 가져왔다면, 이제는 현재 위치에서 가장 가까운 측정소를 실시간으로 탐색해 데이터를 제공합니다.
  • 복수 데이터 소스 통합
    WAQI(세계 대기질 지수), 에어코리아 등 여러 측정 기관의 데이터를 통합 관리합니다. 한 소스에 문제가 생겨도 다른 소스로 자동 전환됩니다.
  • 모델 폴백 개선
    인근에 측정소가 없는 지역도 기상 모델 기반 예측값으로 공기질 정보를 제공합니다. 데이터 공백 지역이 줄었습니다.

기능 추가

  • 홈 화면 위젯 (Android)
    앱을 열지 않아도 홈 화면에서 현재 위치의 미세먼지 수치를 바로 확인할 수 있습니다.
  • 설정 메뉴
    다크 모드 / 라이트 모드 전환, 위젯 사용 여부를 설정 화면에서 관리할 수 있습니다.

개인정보 & 운영

  • 개인정보처리방침 업데이트
    담당자 직통 전화번호를 제거하고, 카카오톡 채널(@soulspectrum)으로 문의 창구를 일원화했습니다.
  • 사업자 정보 정비
